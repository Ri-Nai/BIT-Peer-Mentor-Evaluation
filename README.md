## 🎓 北京理工大学朋辈导师评价系统

📝 用于自动完成对朋辈导师下学生的德育评价工作。

> [!CAUTION]
> 未采用异常处理机制，如有问题请自行调试。
> 大概重启一下程序就好了。

### 🚀 使用方法

1. ⚙️ 配置 `config.json` 文件

```json
{
    "username": "学号",     // 例如：1120234514
    "password": "密码",     // 统一认证密码
    "url": "评价系统网址"  // 已预设，无需修改
}
```

2. 📦 安装必要依赖

```bash
pip install selenium
```

3. ▶️ 运行程序

```bash
python main.py
```
